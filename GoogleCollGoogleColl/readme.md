
### GoogleColl

前言：由于市面上的网址采集器基本均已失效，笔者在进行网站采集时异常麻烦，故此研发本工具。起初只是想做谷歌的网址采集，然却能兼容不同的搜索引擎，需要自写脚本进行配置

说明：内嵌WEBKIT浏览器内核(谷歌同核)，兼容性与谷歌浏览器相似。支持自动翻页、支持自增采集配置。
